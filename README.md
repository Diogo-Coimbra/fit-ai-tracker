# 🏋️‍♂️ Fit AI Tracker

> An AI-powered fitness and nutrition tracker built to optimize gym performance and diet management.

## 🚀 About The Project

**Fit AI Tracker** is a comprehensive full-stack application designed to be a personal, intelligent fitness companion. Unlike standard tracking apps, this project leverages Artificial Intelligence and Computer Vision to generate dynamic workout plans based on available equipment and estimate caloric intake from food images.

## ✨ Core Features (Roadmap)

### 1. Smart Gym Log & AI Personal Trainer
- [ ] **Equipment Database:** A comprehensive library of gym machines and exercises.
- [ ] **LLM-Powered Workouts:** Dynamic workout generation tailored to user goals and specifically filtered by the equipment available to them.
- [ ] **Workout Tracking:** Logging sets, reps, and weights.

### 2. AI Nutrition & Calorie Tracking
- [ ] **Computer Vision Calorie Estimator:** Upload a picture of a meal to get an approximate calorie and macronutrient breakdown.
- [ ] **Daily Dashboard:** Track daily caloric intake vs. goals.

### 3. User Experience & Analytics
- [ ] **OAuth 2.0 Integration:** Secure login (Google).
- [ ] **Progress Analytics:** Time-series charts tracking body weight and lift progressions over time.
